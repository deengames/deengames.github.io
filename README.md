# Adding a New game

- Add the vitals to `data/games.yaml`
- Create the right-sized images in `images`. Right now, these are:
  - `<game name>-featured.png` (450x278)
  - `<game name>-normal.png` (290x179)
- run `rake`.
- Verify results locally. If it works, push.
